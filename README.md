# How to use

```
yarn install
# Run this command to find your cache dir
yarn run rm-cache
# Make note of the command to remove your cache dir and run it
rm -rf /var/folders/x9/80kcdvw148n5glkvr51w7fm80000gn/T/jest
# Now run the test
yarn run test
```
